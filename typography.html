font-family

As a "fallback" system, The font-family attribute can store a number of font names. The next font is tried if the browser can't display the first font.
The list is prioritized from left to right. It will use the first value if it's available, or go to the next one, until the end of the list is reached.

font-family: serif;

The browser will use a Serif font font-family all characters have stroke endings.
