padding

CSS padding properties are used to generate space around an element's content, inside of any defined borders. Shorthand property for padding-top, padding-bottom, padding-right, padding-left

padding: 30px;

Just like the margin, When using 1 value, the padding is set for all 4 sides.

. . .
example

padding: 20px 40px;

When using 2 values:

  • The first value is top/bottom The first value is left/right

. . .
example

padding: 20px 40px 30px 10px;

When using 4 values:

  • The first value is top
  • The second value is right
  • The third value is bottom
  • The fourth value is left
To remember the order , start at the top and go clockwise.
