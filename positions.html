Position

The positioning mechanism used for an element is specified by the position property (static, relative, absolute, fixed, or sticky).

position

Defines the position behavior of the element.

position: static;

The element will remain natural flow

As a result it will not act as an anchor point for the absolutely positioned Child1 element.

position: relative;

The element will remain natural flow It also makes the element positioned Will act as an achor point for the absolutely positined block.
